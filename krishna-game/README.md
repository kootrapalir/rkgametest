# श्री कृष्ण द्वादशी - Divine Journey Game

An immersive 3D interactive game based on the divine verses describing Lord Krishna's beautiful form, written by Kripalu Maharaj.

## Features

- **Interactive 3D Environment**: Explore Krishna's divine form through beautiful 3D scenes
- **Spiritual Journey**: Progress through different aspects of Krishna's beauty
- **Ancient Verses**: Experience the original Sanskrit verses with translations
- **Beautiful UI**: Modern, responsive design with divine aesthetics
- **Audio Experience**: Ambient spiritual music and sound effects
- **Responsive Design**: Works on desktop and mobile devices

## Technologies Used

- **React 18** - Modern React with hooks
- **Three.js** - 3D graphics and rendering
- **React Three Fiber** - React integration for Three.js
- **Framer Motion** - Smooth animations and transitions
- **Tailwind CSS** - Utility-first CSS framework
- **Howler.js** - Audio management
- **Lucide React** - Beautiful icons
- **React Router** - Navigation and routing

## Game Levels

1. **नीलमणि तनु** - Divine Blue Form
2. **मुख छवि** - Beautiful Face
3. **मुकुट शोभा** - Crown Adornment
4. **नेत्र महिमा** - Eyes Glory
5. **चरण कमल** - Lotus Feet

## Installation

```bash
# Clone the repository
git clone <repository-url>
cd krishna-game

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Controls

- **Mouse**: Rotate camera around the scene
- **Scroll**: Zoom in/out
- **Drag**: Pan the camera
- **Arrow Keys**: Navigate between levels
- **Space**: Toggle audio

## Spiritual Significance

This game is based on the ancient verses that describe Lord Krishna's divine form with deep spiritual meaning. Each level represents a different aspect of His beauty and helps devotees connect with the divine through interactive experience.
